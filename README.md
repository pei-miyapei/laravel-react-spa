# laravel-react-spa
[laravel-vue-spa](https://github.com/pei-miyapei/laravel-vue-spa) のReact版です

## 構成

### SPA クライアントコンテナ（SPA フロントエンド）

- Vite
- React 17
- TypeScript
- react-router
- js-pkce
- ant-design（UI フレームワーク）
